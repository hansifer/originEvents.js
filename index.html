Introduction

originEvents.js is a tiny JavaScript library that allows same-origin web pages running within the same browser instance to trigger and handle custom events within and across each other.

demo.html is running in 2 individual iframes below. Click on the buttons to fire events from either page. You can also open demo.html into multiple tabs in your browser for a more convincing demonstration.
